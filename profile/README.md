Standalone external trainer for UNCHARTED: Legacy of Thieves Collection (Uncharted 4 + Lost Legacy) fully compatible with the June 2026 patches and remastered updates. Tested on latest Steam version. Supports single-player campaigns and all difficulties. Enables unrestricted exploration, combat dominance, and treasure hunting without deaths, ammo shortages, or failed climbs. Low overhead, fast attach, stable during long cinematic sequences and intense shootouts.

**Installation Guide**  
1. Download the latest archive.  
2. Extract to a dedicated folder.  
3. Run `UnchartedLegacyTrainer.exe` as administrator.  
4. Launch UNCHARTED: Legacy of Thieves Collection.  
5. Press INSERT to open the trainer interface.  
6. Activate features via hotkeys or menu.  

Re-attach after game restarts or chapter loads.

**Safety & Risk Information**  
- **High risk of save file corruption**: Extreme values on health, ammo, or progression flags can permanently damage your save. Always create multiple backups before every session.  
- Heavy combat or movement modifications may cause temporary physics glitches or animation issues.  
- Rapid changes during autosave cycles frequently result in lost progress or broken story flags.  
- The trainer process commonly triggers antivirus detections due to memory injection.  
- Test all features in a new game first. No recovery possible for corrupted saves.
